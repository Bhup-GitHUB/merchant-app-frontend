# PayApp - Digital Payment Solution

## Overview
A complete digital payment ecosystem. Scan & pay with cashback, send money, and accept merchant payments at the lowest fees.

## Features
- UPI payments
- Cashback rewards
- Merchant gateway
- QR code payments
- Modern UI with React, TailwindCSS, and TypeScript

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack
- React
- TypeScript
- TailwindCSS
- Vite

## Folder Structure
- `src/` - Main source code
- `public/` - Static assets
